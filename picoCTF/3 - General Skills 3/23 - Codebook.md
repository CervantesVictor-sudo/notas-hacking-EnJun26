## Descripción
Run the Python script `code.py` in the same directory as `codebook.txt`.

## Solución

```
VictorCervantes-picoctf@webshell:~$ wget -q https://artifacts.picoctf.net/c/2/codebook.txt
VictorCervantes-picoctf@webshell:~$ wget https://artifacts.picoctf.net/c/2/code.py
VictorCervantes-picoctf@webshell:~$ ls

VictorCervantes-picoctf@webshell:~$ python code.py
```

## Notas adicionales
- Usando el comando 'wget', si le agregamos un '-q', descarga el archivo pero no muestra en la terminal el porcentaje ni la información del archivo, simplemente lo descarga y muestra la siguiente linea para ejecutar comandos.
